# Laravel + Livewire + TailwindCSS + Alpine.js Scaffold

This skill provides a complete setup for Laravel projects with Livewire, TailwindCSS, and Alpine.js integration.

## Features

- ✅ Livewire 3.7 integration
- ✅ TailwindCSS 4.0 with Vite
- ✅ Alpine.js support
- ✅ Admin dashboard with navigation
- ✅ Model traits (Sortable, Searchable, Filterable)
- ✅ Filter system for data queries
- ✅ Value objects (Money, Percentage)
- ✅ CSV writer utilities
- ✅ Laravel Horizon integration
- ✅ Modular admin routing structure
- ✅ Authentication system (login/logout with FormRequest validation)

## Quick Start

When starting a new Laravel project, use this skill to automatically set up:

1. All required dependencies
2. Complete directory structure
3. Core traits and utilities
4. Admin dashboard
5. Layout and components
6. Configuration files
7. Authentication system (login/logout)

## Usage

This skill is designed to be used by AI agents when setting up new Laravel projects. The agent will read the SKILL.md file and create all necessary files and configurations.
